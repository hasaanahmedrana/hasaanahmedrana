## Selected Projects

### 🧠 MindMate AI — Psychiatrist Assistance Platform

A full-stack mental health support platform developed as my final-year project.

- Built a RAG pipeline grounded in **DSM-5-TR** knowledge.
- Used vector search to retrieve relevant context before generating responses.
- Added journal and mood analysis using transformer-based NLP models.
- Developed APIs for conversations, appointments, alerts and patient insights.
- Included safety warnings and escalation support for sensitive situations.

**Technologies:**  
`Python` `Django REST Framework` `PostgreSQL` `pgvector` `LangChain` `React` `React Native`

---

### ⚡ MehfilX — Distributed Real-Time Chat System

A Discord-inspired chat platform designed using separate services and storage systems.

- Implemented real-time communication using WebSockets and Socket.IO.
- Used Redis TTL-based presence tracking for online users.
- Stored structured application data in PostgreSQL and chat messages in MongoDB.
- Used object storage for uploaded media and files.
- Containerized the system using Docker.

**Technologies:**  
`Node.js` `Socket.IO` `Redis` `PostgreSQL` `MongoDB` `Docker` `S3`

---

### 🎯 SkillMatch — AI Resume Matching API

A semantic resume and job-description matching system.

- Converted resumes and job descriptions into vector embeddings.
- Stored and retrieved similar profiles using Pinecone.
- Used Gemini to analyse candidate-job alignment.
- Exposed the matching workflow through a FastAPI backend.
- Designed the pipeline for reusable and structured scoring.

**Technologies:**  
`Python` `FastAPI` `Gemini API` `Pinecone` `RAG` `REST APIs`
